Mar 26 2026 USB 3D 磁力計

本產品是一個可以透過 USB 和電腦連線，並直接將測量結果儲存為 CSV 表格（可由 Excel 直接開啟）的 3D 磁力計。 本產品是基於德州儀器 TMAG5170 霍爾磁力感測 IC，支援 ±300mT/±3000高斯的 3D 測量範圍。 用於連線的電腦端軟體可以直接由 Github 下載，支援 Windows / MacOS / Linux 等作業系統。 我們的連線軟體最高支援每秒紀錄 2500 筆 3D 測量資料。

作者在高中時是 徐有庠盃-臺灣青年學生物理辯論競賽(TYPT) 選手。 當時在高中做實驗時，學校的磁力計因為單價高無法借出實驗室，且測量速度與無法將測量結果紀錄至電腦，造成實驗上許多不便。 因此，作者便開始研究使用霍爾感測器 IC，起初撰寫了 arduino-tmag5170 與 pico-tmag5170 函式庫供學弟妹使用。 在大學時，則利用 STM32 將整個測量電路整合為一個手持儀器，並且開發了用於連線的電腦端軟體，讓沒有程式設計背景的高中生也能簡單上手。

電腦端軟體安裝

如果您是使用 Windows 作業系統，我們提供預先編譯好的執行檔，不需要安裝 Python，請參考 這個影片 的安裝步驟。 如果您是使用 MacOS / Linux 或是想要在 Windows 上以 Python 環境執行，請參考以下的步驟與指令。

首先，電腦端軟體的 Python 檔案可以由 Github 下載。 電腦端軟體需要 Pyside6 與 pyserial 套件才可以執行，建議可以使用 Python 虛擬環境與 pip 安裝。 請在終端機中執行以下三個指令。

python3 -m venv gm_venv
. gm_venv/bin/activate
pip3 install -r python-client/requirements.txt

等待套件安裝完畢後，即可利用以下指令開始執行 USB 3D 高斯計的電腦端軟體。

python3 gauss-meter-v2.py

電腦端軟體使用說明

首先，請選擇序列埠裝置，本產品會顯示為 STM32 Virtual ComPort。 接下來，您可以選擇測量範圍：±75mT、±150mT 或 ±300mT。

本產品有兩個運作模式：即時顯示模式、紀錄儲存模式。 在即時顯示模式下，USB 3D 磁力計所測量到的磁力資料會即時顯示在電腦螢幕上。 在紀錄儲存模式下，USB 3D 磁力計所測量到的磁力資料會紀錄至 CSV 表格中。

紀錄儲存模式可以透過將 □ Save results 打勾開啟，並請選擇存檔的檔案名稱。 軟體預設會儲存 1000 筆 3D 測量資料，如果需要紀錄更多筆測量資料，可以在文字框中打入欲儲存的筆數。 接下來，請選擇紀錄的速度，目前支援的最高取樣頻率為每秒 2500 筆 3D 測量資料。

按下 Start 按鈕即可開始測量。 測量會在達到目標的測量數量之後自動停下，或是可以按 Reset 按鈕強制停止。

達到目標的測量資料數量之後，軟體會產生一個 CSV 檔案，裡面會有四列資料。 第一列為時間軸，單位是微秒 (us)。 後三列為三軸的磁通量密度，單位是毫特斯拉 (mT)。

注意

顯示在選單中的取樣頻率（例如 2500 ksps）僅為估計值，可能會因為 IC 公差而有誤差。 因此，請使用 CSV 檔案中存下的時間軸，以獲得更精確的時間資訊。
